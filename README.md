A simple and interactive command-line application to manage your daily tasks. This project is built using Python to demonstrate basic Object-Oriented Programming (OOP) concepts.

## Features ✨
* **Add Tasks:** Create new tasks quickly.
* **View Tasks:** Check all your pending and completed tasks.
* **Complete Tasks:** Mark tasks as done with a simple status update `[X]`.
* **Simple Interface:** Easy-to-use terminal-based menu.
